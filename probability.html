확률 기초

작성일: 2026. 03. 12

확률론을 처음 정리할 때 자주 마주치는 네 가지 축은 확률과 확률변수, 확률 분포, 기댓값, 우도입니다. 이 문서는 그 흐름을 한 번에 이어서 복습할 수 있도록 구성했습니다.

1. 확률과 확률변수

확률은 어떤 사건이 일어날 가능성을 수치로 표현한 것입니다. 확률변수는 표본공간의 결과를 숫자에 대응시키는 함수이며, 이후 분포와 기댓값을 정의하는 출발점이 됩니다.

핵심 개념

사건의 확률을 해석하려면 먼저 표본공간, 사건, 확률변수의 관계를 분명히 해야 합니다. 확률변수는 이산형일 수도 있고 연속형일 수도 있으며, 이 차이가 이후 분포 표현 방식을 결정합니다.

기본 표현

보통 확률은 $P(A)$처럼 표기하고, 확률변수는 $X, Y$ 같은 대문자로 나타냅니다. 확률변수가 특정 값을 가질 가능성을 다루는 과정이 확률 분포로 이어집니다.

정리 포인트

확률변수는 단순한 숫자가 아니라, 확률적 결과를 수치적으로 다루기 위한 함수라는 점이 중요합니다.

2. 확률 분포

확률 분포는 확률변수가 어떤 값을 얼마나 자주 가질 수 있는지를 체계적으로 설명합니다. 분포를 이해하면 평균, 분산, 사건 확률 같은 다양한 통계량을 계산할 수 있습니다.

이산형과 연속형

이산형 확률변수는 확률질량함수(PMF)로, 연속형 확률변수는 확률밀도함수(PDF)로 표현합니다. 누적된 확률을 확인할 때는 누적분포함수(CDF)를 사용합니다.

이산형 분포

$P(X = x_i)$

연속형 분포

$P(a \le X \le b) = \int_a^b f(x)\,dx$

대표적인 분포 예시

  1. 베르누이 분포: 성공과 실패처럼 두 결과만 갖는 실험을 모델링합니다.
  2. 이항 분포: 베르누이 시행을 여러 번 반복했을 때 성공 횟수를 다룹니다.
  3. 정규 분포: 자연 현상과 통계 추론에서 가장 자주 등장하는 연속형 분포입니다.

3. 기댓값 정리

기댓값은 확률 분포의 중심 경향을 요약하는 대표적인 수치입니다. 단순 평균과 비슷해 보이지만, 각 값에 해당 확률을 가중치로 반영한다는 점에서 확률론적 의미를 가집니다.

계산식

이산형 분포

$\mathbb{E}[X] = \sum_{i} x_i P(x_i)$

연속형 분포

$\mathbb{E}[X] = \int_{-\infty}^{\infty} x f(x)\,dx$

중요한 성질

기댓값의 선형성

변수들이 서로 독립이 아니더라도 $\mathbb{E}[A+B] = \mathbb{E}[A] + \mathbb{E}[B]$가 성립합니다. 이 성질은 복잡한 확률 문제를 단순화할 때 매우 자주 사용됩니다.

4. 우도

우도는 주어진 데이터가 특정 모수 아래에서 얼마나 그럴듯하게 관측되는지를 나타내는 개념입니다. 확률과 비슷해 보이지만, 데이터는 고정하고 모수를 바꿔가며 평가한다는 점에서 방향이 다릅니다.

확률과 우도의 차이

확률은 모수가 주어졌을 때 데이터가 나올 가능성을 말하고, 우도는 관측된 데이터를 기준으로 어떤 모수가 더 적합한지를 비교하는 도구입니다.

$L(\theta \mid x) = P(x \mid \theta)$

왜 중요한가

  1. 모수 추정: 최대우도추정(MLE)의 핵심 기준이 됩니다.
  2. 모형 비교: 어떤 가정이 데이터를 더 잘 설명하는지 판단할 수 있습니다.
  3. 통계 추론: 회귀, 분류, 베이지안 추론과도 깊게 연결됩니다.
