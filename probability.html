[확률론 기초] 모집단부터 베이즈 정리, 확률 분포까지 완벽 정리

작성일: 2026. 03. 12

확률론을 공부하기 위해 가장 먼저 알아야 할 기본 용어들인 모집단, 표본공간, 확률변수부터 베이즈 정리와 이산형·연속형 확률 분포의 개념까지 한 번에 정리합니다.

1. 통계적 추론의 출발점

확률론과 통계학의 출발점은 우리가 알고자 하는 전체 집단과 이를 어떤 확률 모형으로 설명할지를 정하는 것입니다.

모집단, 모수, 그리고 생성 모델

모집단 (Population)

모집단은 정보를 얻고자 하는 전체 대상의 집합입니다. 현실에서는 그 크기가 매우 크거나 무한해서 전체를 직접 아는 것이 보통 어렵습니다.

모수 (Parameter)

모수는 모집단의 확률 분포 형태를 결정하는 값입니다. 실제 분석에서는 직접 관찰하기보다 데이터를 통해 추정합니다.

생성 모델 (Generative Model)

생성 모델은 모수가 주어졌을 때 데이터가 어떤 확률 법칙에 따라 생성되는지를 설명하는 틀입니다.

실제 분석의 방향

실제 문제에서는 모집단 전체를 볼 수 없기 때문에, 표본을 통해 모수를 추정하고 자연의 분포를 잘 근사하는 생성 모델을 세우는 방향으로 분석을 진행합니다.

2. 확률과 확률변수의 수학적 정의

표본 공간과 확률

표본 공간(Sample Space, $\Omega$)은 무작위 실험에서 나올 수 있는 모든 가능한 결과의 집합이며, 확률(Probability, $P$)은 특정 사건이 일어날 가능성을 수치로 표현한 것입니다.

주사위

표본 공간은 $\Omega = \{1, 2, 3, 4, 5, 6\}$ 이고, 짝수가 나올 사건 $A = \{2, 4, 6\}$의 확률은 $P(A) = \frac{3}{6} = \frac{1}{2}$ 입니다.

확률 변수와 확률 분포

확률 변수

확률 변수(Random Variable, $X$)는 표본 공간의 각 결과에 실수 값을 대응시키는 함수입니다. 확률적인 결과를 계산 가능한 형태로 바꿔 주는 역할을 합니다.

DEFINITION
$$X : \Omega \rightarrow \mathbb{R}$$

확률 분포

확률 분포(Probability Distribution)는 확률 변수가 어떤 값을 가질지에 대한 확률 규칙입니다. 즉, 가능한 값과 그 값이 나타날 확률을 연결해 주는 구조입니다.

3. 이산 확률 분포 vs 연속 확률 분포

이산 확률 변수 (Discrete)

확률 변수 $X$가 취할 수 있는 값이 딱딱 끊어져 있는 경우를 말합니다. 이때의 확률 분포는 확률 질량 함수(PMF)로 표현합니다.

PMF PROPERTY
$$p_X(x)=P(X=x), \qquad \sum_x p_X(x)=1$$

누적 질량 함수 (CMF)

특정 값보다 작거나 같은 확률을 모두 더한 함수를 $F_X(x)=P(X \le x)$로 정의합니다.

연속 확률 변수 (Continuous)

확률 변수 $X \in \mathbb{R}$로, 취할 수 있는 값이 연속적인 경우입니다. 이때는 특정한 한 점에서의 확률 대신 확률 밀도 함수(PDF)와 구간 적분으로 확률을 해석합니다.

PDF & CDF
$$\int_{-\infty}^{\infty} f_X(x)dx = 1$$ $$P(a \le X \le b) = \int_a^b f_X(x)dx$$
PDF의 함숫값은 확률 아닙니다

연속 확률 분포에서는 $f_X(x)$의 값 자체가 확률을 의미하지 않습니다. 확률은 반드시 구간의 넓이, 즉 적분값으로 계산해야 합니다.

4. 조건부 확률과 베이즈 정리

조건부 확률 (Conditional Probability)

어떤 사건 $B$가 이미 일어났다는 조건하에 다른 사건 $A$가 일어날 확률을 의미합니다. 표본 공간이 전체 $\Omega$에서 $B$로 축소되는 개념으로 이해할 수 있습니다.

FORMULA
$$P(A \mid B) = \frac{P(A \cap B)}{P(B)} \quad (P(B) > 0)$$

베이지안 정리 (Bayesian Theory)

$P(A \mid B)$를 알고 있을 때 반대 방향의 확률인 $P(B \mid A)$를 계산하기 위해 사용하는 핵심 정리입니다.

BAYES' THEOREM
$$P(B \mid A) = \frac{P(A \mid B)P(B)}{P(A)} \quad (P(A) > 0)$$
베이즈 정리의 구성 요소
  • 사전 확률 (Prior, $P(B)$): 새로운 증거를 보기 전의 확률입니다.
  • 우도 (Likelihood, $P(A \mid B)$): $B$가 주어졌을 때 $A$가 관측될 가능도입니다.
  • 증거 (Evidence, $P(A)$): 관측된 사건 $A$가 전체적으로 발생할 확률입니다.
  • 사후 확률 (Posterior, $P(B \mid A)$): 새로운 증거를 반영한 뒤 갱신된 확률입니다.
주사위 예시

사건 $B$를 짝수, 사건 $A$를 4 이상이라고 두면 $P(B \mid A)$는 4 이상이 나왔을 때 그것이 짝수일 확률로 해석할 수 있습니다. 베이지안 정리는 이런 역방향 확률 계산에 사용됩니다.
