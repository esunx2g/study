확률 기초

작성일: 2026. 03. 12

확률, 통계의 기본 개념부터 베이즈 정리, 그리고 생성 모델의 개념까지 논리적 흐름에 따라 정리한 문서임.

1. 통계적 추론

확률론과 통계학의 출발점은 알고자 하는 전체 집단을 정하고, 이를 어떤 확률 모형으로 설명할지 결정하는 것임.

모집단, 모수, 그리고 생성 모델

모집단 (Population)

정보를 얻고자 하는 전체 대상의 집합. → 보통 알 수 없음
ex) 주사위를 무한히 던졌을 때 나오는 모든 결과

모수 (Parameter)

모집단의 확률 분포 형태를 결정하는 값. → 보통 알 수 없음

생성 모델 (Generative Model)

모수가 주어졌을 때 데이터가 어떤 확률 법칙에 따라 생성되는지 설명하는 틀임.
- 1. 모수가 정해지면 모집단의 확률 분포가 완전히 결정됨.
- 2. 분포를 알면 모집단과 동일한 확률 법칙을 따르는 데이터를 무한히 생성할 수 있음.
- 3. 따라서 생성 모델은 자연의 모집단 확률 분포를 잘 근사하도록 분포의 모수를 학습함.

실제 분석의 방향

현실에서는 전체 데이터를 다 알 수 없기 때문에 일부 표본을 이용하여 전체의 성질을 추측하고 설명하는 모델을 만듦.

2. 확률

표본 공간과 확률

표본 공간 (Sample Space)

무작위 실험에서 나올 수 있는 모든 가능한 결과의 집합.
표기 : $\Omega$

확률 (Probability, $P$)

특정 사건이 일어날 가능성을 수치로 표현한 값.
표기 : $P(A)$

주사위

표본 공간 $\Omega = \{1, 2, 3, 4, 5, 6\}$
짝수가 나올 사건 $A = \{2, 4, 6\}$
확률 $P(A) = \frac{3}{6} = \frac{1}{2}$

조건부 확률과 베이즈 정리

조건부 확률 (Conditional Probability)

어떤 사건 $B$가 일어났다는 조건 하에, 다른 사건 $A$가 일어날 확률임.

FORMULA
$$P(A \mid B) = \frac{P(A \cap B)}{P(B)} \quad (\text{단, } P(B) > 0)$$

베이지안 정리 (Bayes' Theorem)

확률 $P(A \mid B)$를 알고 있을 때 $P(B \mid A)$를 계산하기 위한 정리임.

BAYES' THEOREM
$$P(B \mid A) = \frac{P(A \mid B)P(B)}{P(A)} \quad (P(A) > 0)$$
prior Probability $P(B)$ : 사건 B가 발생할 확률
likelihood $P(A \mid B)$ : 사건 B가 발생했을 때 사건 A가 발생할 확률
evidence $P(A)$ : 사건 A가 발생할 확률
posterior Probability $P(B \mid A)$ : 사건 A가 발생했을 때 사건 B가 발생할 확률
주사위

사건 $A$ = "4 이상": $A = \{4,5,6\}$
사건 $B$ = "짝수": $B = \{2,4,6\}$
$\therefore\; P(A \mid B) = \frac{2/6}{3/6} = \frac{2}{3}$
$\therefore\; P(B \mid A) = \frac{P(A \mid B)P(B)}{P(A)} = \frac{2/3 \,\cdot\, 3/6}{3/6} = \frac{2}{3}$

3. 확률 변수와 확률 분포

확률 변수

확률 변수 (Random Variable)

표본 공간의 각 결과에 실수 값을 대응시키는 함수임.
확률적 결과를 계산 가능한 형태로 바꾸는 역할을 함.
표기 : $X : \Omega \rightarrow \mathbb{R}$

확률 분포(Probability Distribution)

모수로 결정되는 확률 규칙임.
확률 변수가 어떤 값을 가질지에 대한 확률 규칙임.
가능한 값과 그 값이 나타날 확률을 연결하는 구조임.
표기 : $P_X(B) = P(X \in B)$

주사위

확률 변수 $X = \{1, 2, 3, 4, 5, 6\}$ 중 하나의 값
확률 분포 $P(X=x) = \frac{1}{6}$

이산형 (Discrete)

이산 확률 변수 : 이산적인 값 $$X \in \{x_1, x_2, x_3, \cdots\}$$ 이산 확률 분포 : 확률 질량 함수(PMF) $$p_X(x)=P(X=x) → \sum_x p_X(x)=1$$ 확률 : 확률 질량 함수의 y 값 $$P(a \le X \le b) = \sum_{x=a}^{b} p_X(x)$$

PMF (Probability Mass Function)

이산 확률 변수의 분포(확률)를 나타내는 함수가 $p_X(x)$임.
$P(X=x_i) = p_X(x_i), \qquad (i=1,2,\dots,n)$

CMF (Cumulative Mass Function)

이산형 분포에서 확률 변수가 특정 값보다 작거나 같은 확률을 나타내는 함수가 $F_X(x)$임.
$F_X(x) = P(X \le x)$

연속형 (Continuous)

연속 확률 변수 : 연속적인 값 $$X \in \mathbb{R}$$ 연속 확률 분포 : 확률 밀도 함수(PDF) $$f_X(x)=P(X=x) → \int_{-\infty}^{\infty} f_X(x)\,dx=1$$ 확률 : 확률 밀도 함수의 구간 적분값 $$P(a \le X \le b) = \int_{a}^{b} f_X(x)\,dx$$

PDF (Probability Density Function)

연속 확률 변수의 분포를 나타내는 함수가 $f_X(x)$임.
$P(a \le X \le b) = \int_{a}^{b} f_X(x)\,dx$

CDF (Cumulative Distribution Function)

연속형 분포에서 확률 변수가 특정 값보다 작거나 같은 확률을 나타내는 함수가 $F_X(x)$임.
$F_X(x) = P(X \le x)$

PDF와 CDF

(PDF) $f_X(x) = \frac{dF_X(x)}{dx}$
(CDF) $F_X(x) = \int_{-\infty}^{x} f_X(x)\,dx$

PDF의 함숫값은 확률 아닙니다

연속 확률 분포에서 $f_X(x)$의 값 자체는 확률이 아님. 확률 계산 기준은 구간의 넓이, 즉 적분값임.

4. 기댓값 정리

정의

기댓값은 확률 변수에 대해, 그 확률 변수가 따르는 확률 분포를 기준으로 취한 평균임.
각 사건이 벌어졌을 때의 값과, 그 사건이 벌어질 확률을 곱한 것을 전체 사건에 대해 합한 값으로 이해할 수 있음.

수학적 정의

보다 일반적으로는 확률 변수 $X$와 함수 $g(x)$에 대해 $\mathbb{E}_{p(x)}[g(x)]$ 형태로 기댓값을 정의한다. 여기서 $p(x)$는 가중치 역할을 하는 확률 분포이고, $g(x)$는 기댓값을 구하고 싶은 함수임.

GENERAL FORM
$$\mathbb{E}_{p(x)}[g(x)]$$
$p(x)$ : PMF 또는 PDF로 주어지는 가중치
$g(x)$ : 기댓값을 구하고 싶은 함수 ($x$축에서의 함수)

이산 확률 변수

DISCRETE
$$\mathbb{E}_{p(x)}[g(x)] = \sum_x g(x)P(X=x)$$
$g(x)$ : 이산 확률 변수에 대한 함수
$p(x)$ : $P(X=x)$, 즉 확률 질량 함수(PMF)

연속 확률 변수

CONTINUOUS
$$\mathbb{E}_{p(x)}[g(x)] = \int_{-\infty}^{\infty} g(x)f_X(x)\,dx$$
$g(x)$ : 연속 확률 변수에 대한 함수
$p(x)$ : $f_X(x)$, 즉 확률 밀도 함수(PDF)

특징

기댓값은 선형성을 가지기 때문에 복잡한 식을 단순하게 다룰 때 매우 자주 쓰임. 특히 상수항 분리와 합의 기댓값 계산이 직관적으로 정리됨.

동차성

LINEARITY
$\mathbb{E}[aX+b] = a\mathbb{E}[X] + b$

가산성

LINEARITY
$\mathbb{E}[X+Y] = \mathbb{E}[X] + \mathbb{E}[Y]$
핵심

기댓값의 선형성은 보통 독립성과 무관하게 성립함. 그래서 확률 변수의 합이나 변형된 함수의 평균을 다룰 때 기본 도구가 됨.

Monte Carlo

연속 확률 변수 $z \sim p(z)$일 때 함수 $f(z)$의 기댓값은 적분으로 정의됨. 하지만 적분이 어렵다면, 분포 $p(z)$에서 표본을 뽑아 평균으로 근사하는 Monte Carlo 방법을 사용함.

EXPECTATION
$$\mathbb{E}_{z \sim p(z)}[f(z)] = \int f(z)p(z)\,dz$$

샘플 $z^{(1)}, z^{(2)}, \cdots, z^{(N)} \sim p(z)$를 추출하면, 이 기댓값은 표본 평균으로 근사할 수 있음.

MONTE CARLO ESTIMATE
$$\mathbb{E}_{z \sim p(z)}[f(z)] \approx \frac{1}{N}\sum_{n=1}^{N} f\bigl(z^{(n)}\bigr)$$

5. 우도

기본 개념

모수(parameter) $\theta$는 모집단의 확률 분포 형태를 결정하는 값임.
확률 분포(distribution)는 확률 변수가 어떤 값을 가질지에 대한 확률 규칙임.
확률론에서는 $p(x)$를 이미 $\theta$가 주어진 상태의 분포로 봄. 즉, 특정 확률 모형 $\theta$가 주어졌을 때 $x$가 나올 확률을 $p(x \mid \theta)$로 표현함.

Probability (확률)

확률은 분포 $\theta$가 주어졌을 때, 관측값 $(x_1, x_2, \dots)$이 나올 가능성을 보는 관점임. 이때는 $\theta$를 고정하고 $x$를 랜덤하게 봄.

PROBABILITY
$$p(x \mid \theta) = p_\theta(x)$$
주사위 예시

표본 공간: $\Omega = \{1,2,3,4,5,6\}$
파라미터 $\theta$: 공정 주사위라는 확률 분포 고정
관측 사건 $x$: 1부터 6 중 한 개의 숫자
예를 들어 공정 주사위에서 2가 나올 확률은 $\frac{1}{6}$임.

Likelihood (우도)

우도는 관측값 $x$가 이미 주어졌을 때, 어떤 모수 $\theta$가 이 데이터를 더 잘 설명하는지를 보는 관점임. 즉, 이번에는 $x$를 고정하고 $\theta$를 바꿔 가며 비교함.

LIKELIHOOD
$$L(\theta \mid x) := p(x \mid \theta) = p_\theta(x)$$
관측값 $x$가 주어졌을 때, 현재의 모수 $\theta$가 이 고정된 데이터를 만들어냈을 점수처럼 해석함.
주사위 예시

표본 공간: $\Omega = \{1,2,3,4,5,6\}$
파라미터 $\theta_1$: 공정 주사위
파라미터 $\theta_2$: 2가 잘 나오는 주사위
파라미터 $\theta_3$: 확률 분포가 또 다른 주사위
관측 사건 $x=2$가 이미 나왔다고 고정하면, 이제는 어떤 $\theta$가 이 데이터를 가장 그럴듯하게 설명하는지 비교함.

$p(x \mid \theta)$의 관점

같은 식 $p(x \mid \theta)$라도 무엇을 고정하느냐에 따라 해석이 달라짐.
이산 확률 분포에서는 확률로 보면 $p(x \mid \theta)$가 조건부 확률이 됨.
반대로 우도로 보면 $x$를 하나로 고정한 상태에서 PMF 값 또는 PDF 값을 비교하는 관점이 됨.

이산 확률 분포

확률로 보면 $p(x \mid \theta)$는 조건부 확률로 해석함.
우도로 보면 관측값 $x$를 고정한 상태에서 각 $\theta$에 대한 PMF 값을 비교하는 것임.

DISCRETE VIEW
$$P(a \le X \le b) = \sum_x P_X(x)$$
우도 관점에서는 $x$를 1개로 고정하고 $p(x \mid \theta)$ 값을 비교함.

연속 확률 분포

연속 확률 분포에서도 확률로 보면 $p(x \mid \theta)$는 구간 적분으로 해석함.
우도로 보면 특정 관측값 $x$를 고정하고 PDF 값을 비교하는 관점임.

CONTINUOUS VIEW
$$P(a \le X \le b) = \int_a^b f_X(x)\,dx$$
우도 관점에서는 $x$를 1개로 고정하고 $p(x \mid \theta)$ 또는 $f_X(x)$ 값을 비교함.

Maximum Likelihood

최대우도추정은 여러 모수 $\theta_1, \theta_2, \theta_3, \dots$를 비교하면서 현재 데이터를 만들어냈을 점수가 가장 높은 모수를 찾는 방법임. 고정된 데이터를 기준으로 $\theta$를 바꾸어 모집단의 모수를 추정하는 절차임.

MLE
$$\theta^* = \arg\max_{\theta} \log p_\theta(x)$$
해석
  • 비교 대상: 여러 후보 모수 $\theta$들임.
  • 고정 대상: 현재 관측된 데이터 $x$임.
  • 목표: likelihood가 가장 높은 $\theta$를 찾는 것임.
  • 결론: 우도를 최대화하면 모집단의 모수와 가장 유사한 후보를 고르게 됨.

고찰

관측 사건 $x=2$에 대해 $\theta_2$가 가장 높은 우도를 가진다면, 현재 주어진 데이터는 $\theta_2$ 아래에서 나왔을 가능성이 가장 높다고 해석할 수 있음. 즉, 같은 데이터가 주어졌을 때 어떤 모수가 가장 그럴듯한지를 판단하는 기준이 바로 우도임.

활용
  • 모수 추정: 최대우도추정(MLE)의 기본 기준.
  • 모형 비교: 어떤 가정이 데이터를 더 잘 설명하는지 판단.
  • 통계 추론: 회귀, 분류, 베이지안 추론과 자연스럽게 연결.
