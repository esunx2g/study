확률 기초

작성일: 2026. 03. 12

확률론을 처음 정리할 때 자주 마주치는 네 가지 축은 확률과 확률변수, 확률 분포, 기댓값, 우도입니다. 이 문서는 그 흐름을 한 번에 이어서 복습할 수 있도록 구성했습니다.

1. 확률과 확률변수

확률론의 출발점은 전체 대상인 모집단을 어떻게 바라보고, 그 안의 결과를 표본공간과 확률변수로 표현하는지 정리하는 데 있습니다. 이 관계가 분명해야 이후 분포와 기댓값, 우도까지 자연스럽게 이어집니다.

○ Population (모집단)

모집단은 정보를 얻고자 하는 전체 대상의 집합입니다. 실제 문제에서는 모집단 전체를 모두 관측하기 어렵기 때문에, 확률 모형은 이 전체 구조를 간단한 규칙으로 표현하려는 시도라고 볼 수 있습니다.

○ Parameter (모수)

모수는 모집단의 확률 분포 모양을 결정하는 값입니다. 실제 상황에서는 이 값을 직접 알 수 없는 경우가 많고, 표본과 데이터로부터 추정하게 됩니다.

○ Generative Model (생성 모형)

모수가 정해지면 모집단에서 데이터가 생성되는 방식도 함께 정해집니다. 이런 관점에서 확률 모델은 데이터를 만들어 내는 규칙을 가정한 것이라고 볼 수 있습니다.

○ Sample Space (표본공간)

표본공간은 무작위 실험에서 나올 수 있는 모든 가능한 결과의 집합입니다. 보통 $\Omega$로 표기합니다.

$\Omega = \{1,2,3,4,5,6\}$

○ Probability (확률)

확률은 사건이 일어날 가능성을 수치로 표현한 값입니다. 예를 들어 주사위를 던질 때 짝수가 나오는 사건 $A=\{2,4,6\}$에 대해 확률은 다음과 같이 쓸 수 있습니다.

$P(A)=\frac{3}{6}=\frac{1}{2}$

○ Random Variable (확률변수)

확률변수는 표본공간의 결과를 숫자에 대응시키는 함수입니다. 확률적 결과를 계산 가능한 형태로 바꾸는 연결 고리라고 보면 됩니다.

주사위 눈 자체를 확률변수 $X$로 두면, $X$는 1부터 6까지의 값을 가집니다.

예시

동전을 한 번 던져 앞면이면 1, 뒷면이면 0이 되도록 확률변수 $X$를 정의할 수 있습니다. 이렇게 결과를 숫자로 바꾸면 이후 분포와 기댓값을 계산할 수 있습니다.

중요 포인트

확률변수는 단순한 숫자가 아니라, 표본공간의 결과를 수치적으로 다루기 위해 정의한 함수입니다.

2. 확률 분포

확률 분포는 확률변수가 어떤 값을 얼마나 자주 가질 수 있는지를 체계적으로 설명하는 규칙입니다. 이산형과 연속형을 구분해 두면 PMF, CDF, PDF 같은 개념도 훨씬 명확해집니다.

○ 이산형 확률변수

이산형 확률변수는 셀 수 있는 값을 갖는 확률변수입니다. 주사위 눈, 동전의 앞뒷면, 시험의 정답 개수처럼 값이 분리되어 있습니다.

  • - 가능한 값이 떨어져 있으며 셀 수 있습니다.
  • - 보통 $X \in \{x_1, x_2, x_3, \dots\}$처럼 표현합니다.

○ 이산형 확률 분포

이산형 확률 분포는 각 값에 대응하는 확률을 직접 나열하는 방식으로 표현합니다.

○ PMF

PMF는 Probability Mass Function의 약자로, 이산형 확률변수의 각 값에 대한 확률을 나타냅니다.

$p_X(x)=P(X=x), \qquad \sum_x p_X(x)=1$
  • - 각 확률은 0 이상 1 이하입니다.
  • - 모든 값의 확률을 더하면 1입니다.
  • - 구간 확률은 해당 값들의 확률을 더해서 계산합니다.

○ CMF

CMF는 누적된 확률을 나타내는 함수로, 어떤 값 이하가 될 확률을 보여 줍니다.

$F_X(x)=P(X \le x)$
  • - 왼쪽 끝에서는 0에 가깝고 오른쪽으로 갈수록 1에 가까워집니다.
  • - 값이 커질수록 감소하지 않는 함수입니다.
중요 포인트

PMF는 점별 확률, CMF는 그 확률을 누적한 결과입니다. 두 함수는 같은 분포를 다른 관점에서 보여 줍니다.

○ 연속형 확률변수

연속형 확률변수는 어떤 구간 안에서 연속적으로 값을 가질 수 있는 확률변수입니다. 시간, 길이, 온도처럼 측정값을 다룰 때 자주 등장합니다.

  • - 값이 끊기지 않고 이어집니다.
  • - 보통 특정 한 점의 확률보다 구간의 확률을 계산합니다.

○ 연속형 확률 분포

연속형 확률 분포에서는 특정 값 하나의 확률을 직접 주기보다, 밀도 함수를 통해 구간의 확률을 계산합니다.

○ PDF

PDF는 Probability Density Function의 약자로, 연속형 확률변수의 확률 밀도를 나타냅니다.

$P(a \le X \le b)=\int_a^b f_X(x)\,dx$
  • - 밀도 함수 값 자체가 확률은 아닙니다.
  • - 전체 구간에서 적분하면 1이 됩니다.

○ CDF

CDF는 어떤 값 이하가 될 누적 확률을 나타내는 함수입니다. 연속형 분포에서도 동일하게 사용할 수 있습니다.

$F_X(x)=P(X \le x)=\int_{-\infty}^{x} f_X(t)\,dt$
  • - $F_X(-\infty)=0$이고 $F_X(\infty)=1$입니다.
  • - PDF를 적분하면 CDF가 됩니다.
주의

연속형 확률변수에서는 특정 한 점에서의 확률을 직접 해석하지 않습니다. 보통 $P(X=a)=0$으로 두고, 실제 확률은 구간 적분으로 계산합니다.

3. 기댓값 정리

기댓값은 확률 분포의 중심 경향을 요약하는 대표적인 값입니다. 단순 평균처럼 보이지만 각 값에 확률을 가중치로 곱한다는 점에서 확률론적 의미를 가집니다.

○ 계산식

이산형 분포에서는 다음과 같이 계산합니다.

$\mathbb{E}[X] = \sum_{i} x_i P(x_i)$

연속형 분포에서는 다음과 같이 적분합니다.

$\mathbb{E}[X] = \int_{-\infty}^{\infty} x f(x)\,dx$

○ 중요한 성질

중요 포인트

변수들이 서로 독립이 아니더라도 $\mathbb{E}[A+B] = \mathbb{E}[A] + \mathbb{E}[B]$는 성립합니다. 이 선형성은 복잡한 확률 문제를 단순화할 때 매우 자주 사용됩니다.

4. 우도

우도는 주어진 데이터가 특정 모수 아래에서 얼마나 그럴듯하게 관측되는지를 나타내는 개념입니다. 확률과 비슷해 보이지만, 데이터를 고정하고 모수를 바꿔 가며 평가한다는 점이 핵심 차이입니다.

○ 확률과 우도의 차이

확률은 모수가 주어졌을 때 데이터가 나올 가능성을 말합니다. 반대로 우도는 이미 관측된 데이터를 기준으로 어떤 모수가 더 적합한지를 비교하는 도구입니다.

$L(\theta \mid x) = P(x \mid \theta)$

○ 왜 중요한가

  • 모수 추정: 최대우도추정(MLE)의 핵심 기준이 됩니다.
  • 모형 비교: 어떤 가정이 데이터를 더 잘 설명하는지 판단할 수 있습니다.
  • 통계 추론: 회귀, 분류, 베이지안 추론과도 깊게 연결됩니다.
